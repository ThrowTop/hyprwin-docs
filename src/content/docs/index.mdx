HyprWin is a portable Windows tray utility for keyboard-driven and
mouse-assisted window management. Its behavior is configured in
`hyprwin.lua`.

:::note[Independent project]
HyprWin is not affiliated with, endorsed by, or connected to
[hyprwin.cloud](https://hyprwin.cloud/) or its developers.
:::

The documentation is derived from the current HyprWin source code and EmmyLua
annotations.
