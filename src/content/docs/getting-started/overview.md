HyprwinV2 is a Windows-focused window management project with Lua scripting and custom shader support.

This documentation is organized around the main things users need:

- installing and configuring HyprwinV2
- writing Lua scripts
- using the Lua API safely
- authoring custom shaders
- understanding the shader ABI
- building the project from source

Use the sidebar to move between topic areas.
